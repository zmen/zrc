# Zrc - my vim configuration file

My personal configuration for vim. Should work on OSX.

##  Quick installation

- clone this repo to ~/.vim/
- link ~/.vimrc to ~/.vim/\_vimrc

```shell
cd ~/.vim
git clone https://github.com/zmen/zrc.git .
ln -s ~/.vim/_vimrc ~/.vimrc
```
